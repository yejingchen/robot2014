## Robot 2014

本程序用于 2014 年广东省创意机器人大赛，河源中学智能组智能车，可以完成“悬崖勒马”、“红灯停、绿灯行”等功能，但尚未开发完全。

程序目录由 ino 自动生成，如果使用时提示未初始化，请先在一个空目录下运行 `ino init`，然后把本目录合并到该目录。

### ino 使用方式

编译代码：`ino build`
上传代码：`ino upload`
串口监视器：`ino serial`

### 如何安装 ino

请查阅你使用的发行版的软件仓库。如果你使用的没有收录 ino，请访问 [ino 官方网站][1]
